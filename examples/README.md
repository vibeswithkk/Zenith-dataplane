# Zenith AI Examples

This directory contains example scripts demonstrating Zenith AI capabilities.

## Examples

### 1. Basic Usage (`demo_app.py`)

Demonstrates core Zenith engine functionality:
- Engine initialization
- Plugin loading
- Data publishing with zero-copy

```bash
python3 examples/demo_app.py
```

### 2. PyTorch Integration (`pytorch_example.py`)

Shows how to use Zenith with PyTorch for high-performance training:
- ZenithDataset and DataLoader
- Integration with standard training loops
- Performance comparison

```bash
pip install torch
python3 examples/pytorch_example.py
```

### 3. Google Colab Demo (`zenith_demo_colab.ipynb`)

**Recommended for first-time users!**

Interactive Jupyter notebook demonstrating:
- Data loading with Zenith
- Performance benchmarks
- PyTorch training integration
- Multi-format support (Parquet, CSV, Arrow)
- Job scheduling preview

[Open in Google Colab](https://colab.research.google.com/github/vibeswithkk/Zenith-dataplane/blob/main/examples/zenith_demo_colab.ipynb)

### 4. Full Engine Test (`zenith_full_test_colab.ipynb`)

**Complete testing of Zenith engine on Colab!**

Comprehensive notebook that:
- Installs Rust toolchain and compiles from source
- Runs all 103 unit tests
- Benchmarks with multiple dataset sizes
- Tests DataLoader with different batch sizes
- Runs full ML training pipeline
- Generates performance visualizations

Runtime: ~15 minutes on Colab GPU

[Open Full Test in Colab](https://colab.research.google.com/github/vibeswithkk/Zenith-dataplane/blob/main/examples/zenith_full_test_colab.ipynb)

### 5. WASM Plugin Test (`zenith_wasm_test_colab.ipynb`)

**Test WebAssembly plugin system!**

Demonstrates WASM plugin capabilities:
- Compile Rust to WebAssembly (wasm32-wasi)
- Load and analyze WASM modules
- Call plugin functions directly
- Performance benchmarking
- Zenith engine integration

[Open WASM Test in Colab](https://colab.research.google.com/github/vibeswithkk/Zenith-dataplane/blob/main/examples/zenith_wasm_test_colab.ipynb)

### 6. Filter Plugin Demo

The `filter.wasm` file is a sample WASM plugin that filters events.
See `plugins/filter_plugin.rs` for the source code.


## Data

The `data/` subdirectory contains sample datasets generated by the examples.

## Requirements

Basic:
```bash
pip install pyarrow numpy
```

With PyTorch:
```bash
pip install zenith-ai[torch]
# or
pip install torch pyarrow numpy
```

With TensorFlow:
```bash
pip install zenith-ai[tensorflow]
```

### 7. TensorFlow Integration (`tensorflow_example.py`)

Shows how to use Zenith with TensorFlow for high-performance training:
- Zenith data loading concepts
- Integration with TensorFlow datasets
- Performance benefits demonstration

```bash
pip install tensorflow pyarrow numpy
python3 examples/tensorflow_example.py
```

### 8. JAX Integration (`jax_example.py`)

Demonstrates Zenith integration with JAX for numerical computing:
- JAX array handling
- Functional programming patterns
- Gradient computation and optimization

```bash
pip install jax pyarrow numpy
python3 examples/jax_example.py
```

### 9. scikit-learn Integration (`sklearn_example.py`)

Shows traditional ML workflows with Zenith and scikit-learn:
- Data loading and preprocessing
- Random Forest classification
- Feature importance analysis

```bash
pip install scikit-learn pyarrow numpy pandas
python3 examples/sklearn_example.py
```

## Building from Source

If running examples in development mode:

```bash
# Build Rust core first
cd /path/to/Zenith-dataplane
cargo build --release

# Then run examples
python3 examples/demo_app.py
```
